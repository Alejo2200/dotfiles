# Configuración de wlogout
Esta carpeta contiene los archivos de configuración para la interfaz de apagado/cierre de sesión que aparece al hacer clic en el botón de la Waybar.

- **layout**: Define los botones y acciones (Lock, Shutdown, etc.).
- **style.css**: Define la apariencia (colores, bordes, espaciado) de la ventana emergente.
